HTML5 Semantic Elements

Introduction

The aim of this site is to clarify the use of some of the newer HTML5 semantic tags so that I know when to use them when creating sites. Semantic elements were added in HTML5 and are considered as such as they accurately describe both the purpose of the element and the type of content inside them.

Layout Elements

These are the main elements used when laying out your webpage using HTML5.

Headings & Navigation

Element Use Example
header <header>...</header> This is used to contain the header of the webpage.
nav <nav>...</nav> This is used to contain the navigation section of a website.
section <section>...</section> This is used to define a section in a document.
article <article>...</article> This is used to define a section in a document.
aside <aside>...</aside> This is used to define independent, self-contained content.
footer <footer>...</footer> This is used to define the footer for a document or section.

Style Elements

These are the main elements used when styling your webpage using HTML5.

Text Styling

Element Use Example
emphasis <em>...</em> This is used to mark text to stress emphasis.
strong <strong>...</strong> This is used to indicate contents that have strong importance, seriousness or urgency.
sub <sub>...</sub> This is used to create subscript text.
sup <sup>...</sup> This is used to create superscript text.
code <code>...</code> This is used when highlighting <code snippits> within your page.
mark <mark>...</mark> This is used when highlighting marking text of importance.
cite <cite>...</cite> This is used when defining the title of a creative work within your page.
